Latitude plotted against various weather factors

Weather patterns as a function of Latitude
Pandas and Seaborn

In efforts to better understand weather patterns across the globe, our humble group of data sleuths took to a popular weather API (OpenWeatherMap API) and decided to plot key weather indicators against Latitide which runs up and down on the globe.

By utilizing the OpenWeatherMap API and citipy library, we created python scripts aimed at randomly selecting 500 cities around the globe and pulling weather data including Cloudiness, Humidity, Max Temp and Wind Speed. Then, using Seaborn, plotting the data against Latitude and analyzing for relationships. Some of the limitiations of the analysis include but are not limited to a) we did not pull in longitude which provides additional dinemsionality to the analysis; b) weather may not behave in a linear fashion as depicted in the following plots with a linear regression line of best fit; c) there are better visuals for weather across the globe (i.e. map vs x,y plot).
